  • HLSPlaylistParser

    A parser for HLS (HTTP Live Streaming) playlist files that extracts track information.

    This class provides functionality to download and parse HLS master playlists (.m3u8 files), extracting information about available video, audio, and subtitle streams. It also handles DoveRunner DRM-related custom information if present in the manifest.

    Use this class to:

    • Download HLS manifest files from a URL
    • Parse manifests to extract media stream information
    • Access structured information about available tracks and their properties
