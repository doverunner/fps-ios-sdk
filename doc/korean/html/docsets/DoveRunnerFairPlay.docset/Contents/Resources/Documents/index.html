Version 2.5.0

What’s New in Version 2.5.0

Changed 제품 브랜드명이 DovrRunner로 변경됩니다.

  • DovrRunner는 PallyCon의 새로운 이름입니다.
  • 기존 PallyCon 콘텐츠 보안 서비스와 관련된 다양한 제품의 이름을 DovrRunner로 변경합니다.
  • 이 기간 동안 Docs 사이트의 일부 문서 및 코드에서 PallyCon과 DovrRunner를 혼용하여 사용할 수 있습니다.

Changed API에 PallyCon prefix가 제거됩니다.

Note

  • 일부분 Deprecated API 가 제거됩니다.
  • API 명이 변경되었으므로 샘플 프로젝트를 참고하여 적용하세요.

Version 2.4.0

What’s New in Version 2.4.0

Changed SDK 최소 지원 버전이 iOS 13.0으로 변경됩니다.

Changed PallyConDrmConfiguration.renewalInterval 운영 방식이 변경됩니다.

  • license renewal 을 사용할 때 콘솔에 설정한 값으로 PallyConDrmConfiguration.renewalInterval 에 반드시 설정해줘야 합니다.
    • 콘솔에서 renewal 설정 없이 PallyConDrmConfiguration.renewalInterval 설정만으로 동작하지 않습니다.
    • 추가적인 궁금증이 있다면 헬프데스크로 문의주시기 바랍니다.

Added PallyConOfflineExpiryDate 가 추가됩니다.

  • 오프라인 라이선스 만료 정보 클래스가 추가됩니다..
  public class PallyConOfflineExpiryDate {
    /// Offline license downloaded date.
    public private(set) var offlineLicenseDownloadedDate: Date?
    /// Rental expiry date: `offlineLicenseDownloadedDate` + `rentalDuration`
    public private(set) var rentalExpiryDate: Date?
    /// Playback expiry date: `offlineLicenseDownloadedDate` + `playbackDuration`
    public private(set) var playbackExpiryDate: Date?
    /// Rental duration in seconds
    public private(set) var rentalDuration: String?
    /// Playback duration in seconds
    public private(set) var playbackDuration: String?
  }

Added getOfflineLicenseExpiryDate 가 추가됩니다.

  • 입력한 Content ID의 만료 정보를 반환합니다.
  func getOfflineLicenseExpiryDate(find contentId: String) -> PallyConOfflineExpiryDate

Added HLSTracksPlaylistParser 클래스가 추가됩니다.

  • HLSTracksPlaylistParser 클래스는 m3u8 파일에서 다운로드 받을 Track 정보를 제공합니다.
  • `HLSTracksPlaylistParser.HLSManifest’ 를 통해 Track 정보를 알수 있습니다.
    • Advanced Sample을 참고하세요.

Remove PallyConDrmConfiguration.keyIdList 가 제거됩니다.

  • 라이선스 요청 시 keyIdList 입력 없이 가능합니다.

Remove PallyConDrmConfiguration.allowsKeyRotation 이 제거됩니다.

  • AuthData 를 이용하여 제어할 수 있습니다.

Version 1.14.0

What’s New in Version 1.14.0

Added AVAssetResourceLoaderDelegate.resourceLoader() 의 내용을 외부에서 변경할 수 있는 resourceLoaderRequest() 함수가 추가됩니다.

  • 재생하려는 playlist.m3u8 의 내용을 수정할 수 있습니다(자막 추가 같은).
  @objc public protocol PallyConFPSLicenseDelegate: NSObjectProtocol {
    /**
       You can modify the `requestResource` of `AVAssetResourceLoaderDelegate.resourceLoader()` used within PallyCon SDK as needed.
       (e.g. add subtitles to the m3u8 content you want to play)
       The `resourceLoaderRequest()` function is called when `PallyConFPSSDK.mainm3u8Scheme` is applied to the m3u8 URL.

       Example:
        // Changed [https] URL Scheme to [PallyConFPSSDK.mainm3u8Scheme]
        https://somedomain.com/playlist.m3u8 -> mainm3u8://somedomain.com/playlist.m3u8

       @param requestResource
              An instance of AVAssetResourceLoadingRequest that provides information about the requested resource.
       @result TRUE  if the delegate can load the resource indicated by the AVAssetResourceLoadingRequest; otherwise FALSE.
       */
    optional func resourceLoaderRequest(_ requestResource: AVAssetResourceLoadingRequest) -> Bool
  }

  • m3u8주소 URL Scheme(https) 을 PallyConFPSSDK.mainm3u8Scheme(mainm3u8)으로 변경해야 resourceLoaderRequest() 함수가 호출됩니다.

  • resourceLoaderRequest() 함수 구현 예제는 샘플 프로젝트(advanced sample)에서 확인할 수 있습니다.
