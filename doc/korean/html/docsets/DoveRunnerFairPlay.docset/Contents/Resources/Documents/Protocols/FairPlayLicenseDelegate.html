  • resourceLoaderCallback(requestResource:)

    You can modify the requestResource of AVAssetResourceLoaderDelegate.resourceLoader() used within SDK as needed. (e.g. add subtitles to the m3u8 content you want to play) The resourceLoaderCallback() function is called when DoveRunnerFairPlay.mainm3u8Scheme is applied to the m3u8 URL. Example:

    // Changed [https] URL Scheme to [DoveRunnerFairPlay.mainm3u8Scheme]
https://somedomain.com/playlist.m3u8 -> mainm3u8://somedomain.com/playlist.m3u8

    @param requestResource An instance of AVAssetResourceLoadingRequest that provides information about the requested resource. @result TRUE if the delegate can load the resource indicated by the AVAssetResourceLoadingRequest; otherwise FALSE.
