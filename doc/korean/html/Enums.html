  • RenewalInterval

    License Renewal Interval. It must be the same as the time(interval) specified in the DoveRunner console. If it is longer than the specified time, a license expiration error occurs, and if it is smaller than the specified time, the license cost will increases. The value received from the DoveRunner license server takes precedence over the API parameter value.
