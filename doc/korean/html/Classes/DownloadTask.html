DownloadTask

@available(iOS 11.0, *)
@objc
public class DownloadTask : NSObject

A class that receives remote AVURLAssets and downloads them locally. You can create the instance by using (DoveRunnerFairPlay object).createDownloadTask(config:downloadCellerAccess:isMakeAssetDownloadTask:downloadMinimumBitrate:delegate:). You can also cancel current download using the same instance.

Methods

  • resume()

    Starts downloading content. Previous download can be resumed if this method is called after cancel() method without terminating the application. You need to check network availability before calling this method.
