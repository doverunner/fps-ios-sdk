DoveRunnerFairPlay SDK for iOS Document

DoveRunnerFairPlay SDK makes it easy to apply Apple’s FairPlay Streaming(FPS) DRM when developing media service apps for iOS. This document describes how to use the libraries and sample projects included in the SDK.

A description of the DoveRunner Multi DRM service that works with FPS clients is provided in a separate online guide. For technical questions about using the SDK, please visit the Helpdesk site.

Supported Environments

  • iOS 13 or later

Note:

  • This SDK does not work on Simulator.

  • The following items are needed to test your FPS HLS contents:

    • You need to acquire FPS Deployment Package from Apple, and send us FPS Certification file, Private Key file(RSA) and FPS App Secret key(ASK) which are included in the package.

  • If the above conditions are not met, DoveRunnerFairPlay Sample can not play FPS HLS content.

NOTE:

  • Once the keys received from APPLE are registered on the DoveRunner site, you need to encrypt the HLS content with Site ID and Site Key in the provided packager.
  • If you need help, please issue a ticket on Helpdesk site.

Configuring your project

You can add DoveRunnerFairPlay SDK by following steps:

  1. Extract DoveRunnerFairPlay.zip file.

  2. Add the DoveRunnerFairPlay.xcframework library provided in the lib folder to your project.

    • Add the framework to the Embedded Binaries setting in Xcode General tab.
    • Add the framework’s path to Framework Search Paths setting in Xcode Build Settings tab.
    • On the Build Settings tab on the TARGET, set the Always Embed Swift Standard Libraries to Yes.

  3. Bitcode settings

    • iOS library does not support bitcode.
      • In the project TARGETS, set the Build Settings tab Enable Bitcode to NO.
    • The tvOS library supports bitcode.

NOTE:

  • Advanced Sample requires GoogleCast.framework.
  • Install(pod intstall) the GoogleCast.framework using COCOAPODS and run the generated FairPlayAdvanced.xcworkspace.

Quick Start

The following code shows how to stream or download FPS HLS content using DoveRunnerFairPlay SDK.

Streaming FPS content

// 1. initialize a DoveRunnerFairPlay SDK.
let doverunnerSdk = DoveRunnerFairPlay()

let urlAsset = AVURLAsset(url: "Content Url(m3u8)")
let config = FairPlayConfiguration(avURLAsset: urlAsset, 
                                   contentId: "ContentID", 
                                   certificateUrl: "Certification URL", 
                                   authData: "Token or CustomData")

// 2. Before playing FPS content, prepare to acquire a license by setting token information.
doverunnerSdk.prepare(drm: config)

Downloading FPS content

// 1. Create DoveRunnerFairPlay instance.
let doverunnerSdk = DoveRunnerFairPlay()

let urlAsset = AVURLAsset(url: "Content Url(m3u8)")
let config = FairPlayConfiguration(avURLAsset: AVURLAsset, 
                                   contentId: "ContentID", 
                                   certificateUrl: "Certification URL", 
                                   authData: "Token or CustomData")

// 2. To download FPS content, create a DownloadTask object and enter token information to prepare for license acquisition.
let downloadTask = doverunnerSdk.createDownloadTask(drm: config, 
                                         downloadDelegate:"FairPlayDownloadDelegate")

// 3. Start downloading content
// If the content download was canceled previously, it will be resumed.
downloadTask.resume()

// 4. Cancel download (if needed)
downloadTask.cancel()

Acquiring a CustomData for play the chromcast

// 'DoveRunnerFairPlay SDK' supported chromcast.
// Get CustomData needed to create 'GCKMediaInformation' object when using Chrome Cast SDK
// CustomData should be created by DoveRunnerFairPlay instance.
let jsonData = doverunnerSdk.getJsonforChromecastPlayback(authData: "Token or CustomData")
